🔗 Conectar Google Sheets

Primero necesitas crear una hoja de Google para guardar los datos.

📋 Pasos para configurar:

  1. Abre Google Sheets y crea una hoja nueva
  2. Ve a Extensiones → Apps Script
  3. Borra todo el código y pega el código que aparece abajo
  4. Haz clic en Implementar → Nueva implementación
  5. Selecciona "Aplicación web"
  6. En "Quién tiene acceso" elige "Cualquier persona"
  7. Copia la URL y pégala aquí abajo
